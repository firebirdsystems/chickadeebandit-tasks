# Tasks

A [Chickadee Bandit](http://chickadeebandit.com) app.

Full-featured family task manager. Create tasks with due dates, priorities, labels, recurrence, and subtasks. Assign to family members. Also syncs with Google Tasks and Apple Reminders.

## Features

- Create tasks with due dates, priorities (urgent / high / normal / low), and labels
- Recurring tasks with flexible schedules
- Subtasks and task dependencies
- Assign tasks to any family member
- Syncs with Google Tasks and Apple Reminders via hub settings
- Filter and sort by assignee, priority, due date, or label

## Install

In your hub, go to **Apps → Install from URL** and paste:

```
https://github.com/firebirdsystems/chickadeebandit-tasks/releases/latest/download/bundle.json
```
